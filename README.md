# Renewable Energy Generation Card

Animation inspirée du [motion de Tanjim sur X](https://x.com/tanjim38/status/2015288620312154520?s=46&t=oQmISEMIEdgUO_t0HNANhw), refaite avec la stack ci-dessous.

## Stack

- **React 19** + **TypeScript** + **Vite** — app et build
- **Tailwind CSS v4** — styles (via `@tailwindcss/vite`)
- **Motion** — animations (séparateurs, count-up, transitions)
- **HugeIcons** — icônes
- **clsx** + **tailwind-merge** — utilitaires CSS

## Scripts

```bash
npm run dev     # serveur de dev
npm run build   # build de prod
npm run preview # prévisualisation du build
```
